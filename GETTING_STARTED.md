Welcome to Python from Scratch! This guide will help you get started with your Python learning journey.

## Quick Start (5 Minutes)

### Step 1: Clone the Repository
```bash
git clone https://github.com/syedalamshah/Python-fromScratch.git
cd Python-fromScratch
```

### Step 2: Verify Python Installation
```bash
python --version
# or
python3 --version
```
You should see Python 3.x. If not, [download Python here](https://www.python.org/downloads/).

### Step 3: Run Your First Example
```bash
python 01_basics/01_variables.py
```

You should see output explaining variables and data types!

## How to Study Each Topic

### 1. Read First
Open the Python file and read through the comments and code. Don't run it yet!

### 2. Predict Output
Try to predict what the code will output before running it.

### 3. Run the Code
```bash
python filename.py
```
Compare actual output with your prediction.

### 4. Experiment
Modify the code:
- Change variable values
- Add new examples
- Break things intentionally to see error messages

### 5. Take Notes
Keep a notebook (digital or physical) with:
- Key concepts
- Syntax patterns
- Common mistakes
- "Aha!" moments

### 6. Practice
Do the exercises without looking at solutions first.

## Common Issues

### "Python not found"
- Make sure Python is installed
- Add Python to PATH
- Try `python3` instead of `python`

### "Module not found"
- This repository uses only built-in modules
- No additional installation needed
- If you see this, check for typos

### "Permission denied"
- On Linux/Mac, you might need: `chmod +x filename.py`
- Or always use: `python filename.py`
