## Getting Started

### Prerequisites

```bash
# Python 3.x (3.8+ recommended)
python --version

# Code editor (VS Code, PyCharm, Sublime Text)
```

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/syedalamshah/Python-fromScratch.git
cd Python-fromScratch

# 2. Create virtual environment (recommended)
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt
```

### Usage

```bash
# Navigate to any topic and run examples
cd basics
python variables_and_types.py

# Run tests (if available)
pytest tests/
```
