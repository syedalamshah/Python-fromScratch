## üöÄ Getting Started

### Prerequisites
- Python 3.x installed on your computer
- A text editor or IDE (VS Code, PyCharm, etc.)
- Basic understanding of using a terminal/command line

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/syedalamshah/Python-fromScratch.git
   cd Python-fromScratch
   ```

2. Run any Python file:
   ```bash
   python 01_basics/01_variables.py
   ```

## üí° How to Use This Repository

### For Complete Beginners
1. Start with `01_basics/01_variables.py`
2. Read the comments and explanations in each file
3. Run the files to see output: `python filename.py`
4. Modify the code and experiment
5. Complete exercises in `08_exercises/`

### For Each Topic
1. **Read** - Go through the code and comments
2. **Run** - Execute the file to see results
3. **Modify** - Change values and see what happens
4. **Practice** - Try the exercises
5. **Create** - Build your own examples

### Running Examples
```bash
# Run a specific file
python 01_basics/01_variables.py

# Navigate to a directory and run
cd 02_control_flow
python 01_conditionals.py
```
